## 1. 功能亮点 (Feature Highlights)

AlphaDash 不仅仅是一个记账工具，它是为您量身定制的"交易员驾驶舱"。

### 🛡️ 隐私优先 (Privacy First)
- **本地模式**：默认情况下，您的所有数据仅存储在浏览器的 LocalStorage 中，**不上传任何服务器**。您的资产信息只有您自己知道。
- **云端同步**：当且仅当您注册并登录后，数据才会加密存储至 Google Cloud，方便您在多台设备间无缝切换。

### 🚀 极速录入 (Efficiency)
- **Excel 批量粘贴**：支持从 Excel 直接复制数据粘贴到网页，一键解析几十个账号的日记账。
- **智能计算**：只需输入"今日余额"，系统自动根据昨日余额计算出"磨损"和"净利润"，告别繁琐的手工计算。

### 📊 专业看板 (Pro Dashboard)

- **全维度数据**：
  - **ROI 分析**：实时计算本月投资回报率。
  - **净值曲线**：自动生成 30 天及历史累计净值走势图。
  - **热力图**：像 GitHub 代码提交一样的"打卡热力图"，直观展示您的交易勤奋度。

### 📱 社交分享 (Social Growth)

## 2. 操作指南 (User Guide)

### 第一步：账号设置
1.  **添加/删除账号**：
    - 点击看板中间的 `+ 添加账号` 按钮即可新增空账号。
    - 点击 `- 删除账号` 可移除最后一个账号。
    - *提示：普通用户限制 1 个账号，解锁 Pro 后可管理更多（目前演示期上限为 50 个）。*

### 第二步：日常数据录入
您有两种方式录入每日数据：

#### 方式 A：每日手动打卡（推荐）
1.  在页面下方的 **"每日数据录入"** 区域。
2.  可以直接修改表格中的数字（积分、今日余额等）。
3.  点击右下角的 **"保存"** 按钮。

#### 方式 B：批量导入（Power User）
1.  点击右上角的 **"文件导入"** 图标（文件夹形状）。
2.  在您的 Excel 中选中包含数据的区域（需包含日期、账号、积分等列）。
3.  **Ctrl + C** 复制，然后在网页弹窗中 **Ctrl + V** 粘贴。
4.  点击 **"解析数据"** -> **"确认导入"**。

### 第三步：查看报表
- 点击顶部的 **"报表" (Pie Chart)** 图标，进入深度分析模式。
- 您可以按月查看详细的收支明细表。

### 第四步：数据备份与导出
- 点击右上角的 **"导出"** 按钮，支持下载为 CSV (Excel) 或 JSON 格式，确保数据拥有权归您所有。

## 3. 注意事项 (Important Notes)

### ⚠️ 关于数据安全
1.  **游客 (未登录)**：
    - 数据保存在**当前浏览器的缓存**中。
    - **切勿**清除浏览器缓存，否则数据会丢失！
    - 建议定期导出 CSV 备份。
2.  **注册用户**：
    - 数据保存至云端数据库，永久存储。
    - 即使清除缓存或更换电脑，登录后数据即可自动恢复。

### 💡 磨损计算逻辑
AlphaDash 使用以下公式自动计算每日磨损：
> **磨损 = 昨日余额 + 充值/调整 - 今日余额 - 今日收益**

如果计算出的磨损为负数（即钱变多了），系统会自动将其归类为"未记录的收益"。
